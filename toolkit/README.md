# Take-home toolkit

The reusable kit built during the workshop. Contents land here during the prep week and the day itself:

- `AGENTS.md` starter — global rules (incl. the no-scaffolding-trap SOLID rule)
- `hooks/` — PreToolUse guard + Stop-runs-tests
- `skills/` — one worked skill + the self-improving-skill recipe
- `orchestrator/` — orchestrator + reviewer/critic subagents with the Repeat-Until-Good loop
- `spec-templates/` — constitution / spec / plan / tasks + Given-When-Then acceptance template
- `checklists/` — spec-gate, AI-test-review, context-budget, legacy-modernization, 30/60/90
- `legacy-playbook/` — characterization tests (Verify), tSQLt template, strangler-fig/YARP, Azure DevOps MCP config, TFVC→Git
