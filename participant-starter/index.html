Résztvevői starter

Egy szándékosan kicsi Next.js projektet kapsz. Az alkalmazás elindul és vannak ellenőrzési parancsai, de a csapatod működési szabályait, specifikációját és későbbi szolgáltatásait még neked kell hozzáadnod.

Lényeg: a starter technikai alap, nem kész fejlesztési rendszer és nem kész termék.

Ami már készen áll

  • Next.js App Router, TypeScript és Tailwind alap.
  • Egy kicsi, futtatható példaoldal és mintateszt.
  • typecheck, lint, test és build parancs.
  • Kiinduló repo- és dizájnszabályok, amelyeket a saját projektedhez igazítasz.

Ami szándékosan nincs benne

  • Nincs kész üzleti feature vagy előre eldöntött specifikáció.
  • Nincs adatbázis, API-kapcsolat vagy valódi környezeti titok.
  • Nincs helyetted elfogadott repo-szabályrendszer.
  • Nincs automatikus bizonyíték arra, hogy a csapatod munkamódja működik.

Első biztonságos lépések

Előfeltétel: a starter már a saját, írható repódban van, és Node.js 22 fut a gépeden.

  1. Ellenőrizd a környezetet.
    node --version
npm --version

    Elvárt eredmény: mindkét parancs verziószámot ír ki.

  2. Telepítsd a zárolt függőségeket.
    npm ci

    Elvárt eredmény: a parancs hibakód nélkül befejeződik. Ne másolj titkot az .env.example fájlba.

  3. Futtasd a teljes helyi preflightot.
    npm run typecheck
npm run lint
npm run test
npm run build

    Elvárt eredmény: mind a négy parancs sikeres. A teszt jelenleg egy szándékosan kicsi mintát ellenőriz.

  4. Olvasd el, majd igazítsd a saját repohoz az AGENTS.md és DESIGN-GUIDELINE.md fájlokat. Ne tekintsd késznek az általános kiinduló szöveget.
  5. Állj meg a feature előtt. A következő modulban rögzítsd a repo célját, határait és kötelező ellenőrzési pontjait.

Sikerfeltétel: egy friss agent chatelőzmény nélkül meg tudja mondani, hol dolgozhat, mely parancsokat kell futtatnia, és mely döntéseket nem hozhatja meg helyetted.

Ha elakadsz: hálózati vagy telepítési hiba esetén használd a tréner SHA-hoz kötött, kitalált adatokkal készült preflight-eredményét. Jelöld REPLAY-ként; ne állítsd, hogy helyben lefutott.
